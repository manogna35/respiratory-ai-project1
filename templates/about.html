About Our Project

AI Powered Respiratory Health Sound Analyzer for smart and early disease detection.

Project Overview

This intelligent healthcare system analyzes breathing, coughing, and wheezing sounds to identify possible respiratory diseases using Artificial Intelligence and Deep Learning.

Why This Project?

Respiratory diseases are common and sometimes detected late. This project helps in early screening, faster diagnosis support, and remote monitoring.

Key Features

  • Easy audio upload
  • Instant prediction results
  • Confidence score display
  • User-friendly interface
  • Supports early detection

Dataset Used

ICBHI Respiratory Sound Dataset containing real patient breathing recordings. Data augmentation was used to improve model performance.

Features Extracted

  • MFCC
  • Mel Spectrogram
  • Spectral Contrast
  • Zero Crossing Rate

System Workflow

  • Upload Audio File
  • Convert to Spectrogram
  • Extract Audio Features
  • Predict Disease
  • Display Confidence Score

Diseases Detected

  • Asthma
  • COPD
  • Pneumonia
  • Bronchiectasis
  • Healthy Condition

Advantages

  • Remote patient monitoring
